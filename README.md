# 🐍 Python Data Tools

Collection of Python projects including data processing scripts and a simple REST API built with FastAPI.

---

## 🚀 Features

- 📊 CSV data analyzer
- ⚡ FastAPI backend service
- 🧪 Simple in-memory data storage
- 📦 Clean and readable code structure

---

## 📁 Project Structure
```
├── csv_analyzer.py # Script for analyzing CSV data
├── data.csv # Sample dataset
├── api.py # FastAPI application
├── requirements.txt # Dependencies
└── README.md
```

## ⚙️ Installation

```bash
git clone https://github.com/bdshkaaa/python-data-tools.git
cd python-data-tools
pip3 install -r requirements.txt
